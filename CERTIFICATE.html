Formal statement

I, Mughirah Nasir, certify that I am the originator and sole author of this project. I conceived the problem framing, designed the four-layer resolution architecture, wrote the phonetic encoding algorithm, curated every entry in the canonical lexicon, variant map, and benchmark dataset, designed the homograph guard, wrote all 162 tests including the regression and adversarial suites, designed the production hardening, and produced every piece of documentation in this repository.

Items authored personally
  • Four-layer resolution pipeline (phrase → variant → phonetic → unchanged)
  • Multi-token phrase rewrite layer — 125+ curated compound forms
  • Per-token confidence scoring (0.0–1.0) preserving never-silently-guess
  • Phonetic encoding algorithm — every digraph rule and vowel-class choice
  • Curated canonical lexicon — 655 words across 13 POS categories
  • Curated variant map — ~430 SMS shorthand entries from real Pakistani usage
  • Homograph guard — 6 registered ambiguous-pair groups
  • "Never silently guess" rule — design constraint, not afterthought
  • 250-example hand-curated gold-standard benchmark dataset (50+ categories)
  • Adversarial perturbation generator producing 242 more eval examples
  • Benchmark scoring harness, latency suite, and 4-baseline comparison study
  • TF-IDF char n-gram ML baseline (sklearn-trained, cosine similarity)
  • Both regression-tested bugs, found by feeding the live demo real text
  • All 162 tests across 8 test files (incl. 27 multitoken + 35 adversarial + 13 client SDK)
  • Python client SDK with zero third-party deps, retries, batch chunking
  • Production hardening: CORS allowlist, request-size limit, rate limit, /metrics endpoint
  • One-click Render.com deploy (render.yaml Blueprint)
  • Search recall integration showcase (examples/search_recall_demo.py)
  • API contract — endpoint shapes, batch semantics, error responses
  • Dark editorial frontend — typography, color tokens, offline fallback
  • All documentation including this certificate, DESIGN.md, and four docs/ markdown files
