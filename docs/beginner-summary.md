# Beginner Summary

This project is a Roman Urdu spelling normalizer.

In simple words, it cleans messy Pakistani Roman Urdu text and turns common spellings into one standard form.

Examples:

- bht, bohat, bhot become bahut
- nhi, nai, nahin become nahi
- yr and yar become yaar
- kr rha becomes kar raha

This is useful before search, chatbots, sentiment analysis, or other NLP systems because those systems work better when the spelling is consistent.

The project uses a FastAPI app, a command line tool, tests, examples, benchmarks, and documentation.
