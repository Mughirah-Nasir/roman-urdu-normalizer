Roman Urdu spelled every which way — normalized to one canonical form.

Pakistani Roman Urdu on WhatsApp and Twitter is a spelling free-for-all. "kya / kia / kyaa", "bht / bohat / bahut", "nhi / nahin / nai" all mean the same thing. This API resolves them — through a four-layer pipeline that knows when to guess and when to stay quiet.

02

How it resolves a word

Layer 1
Variant map

An exact-match table for SMS shorthand the phonetic algorithm can't reach because it drops too many vowels: bht → bahut, nhi → nahi. O(1) dict lookup. Highest priority.

Layer 2
Phonetic key

Each word is reduced to a sound-skeleton (digraph folding, vowel-class collapse, repeat squashing). kya / kia / kyaa share one key. A pre-built index maps the key to canonical lexicon words.

Layer 3
Unknown

Nothing matched. The token passes through unchanged and is flagged source: unknown. We never silently guess — silent guessing is how normalizers poison every system downstream.
